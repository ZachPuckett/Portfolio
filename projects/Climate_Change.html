Global CO₂ Emissions and Pollution Mapping

This project uses global emissions and pollution data to explore environmental disparities across space using mixed symbol, orthographic, and proportional symbol mapping.

Map 1: Mixed Symbol Map – Cumulative vs 2019 CO₂ Emissions

This map combines a choropleth layer of cumulative CO₂ emissions with a dot density overlay of 2019 emissions. The map uses an equal-area projection to preserve spatial density and reveal long-term contributors versus current emitters.

Map 2: Orthographic Map – PM2.5 Exposure in Africa

This map shows average exposure to PM2.5 air pollution using an orthographic projection centered on New Delhi. The custom projection highlights regional health risks from air pollution.

Map 3: Regional CO₂ Emissions – Proportional Symbol Mapping

This proportional symbol map displays cumulative emissions by region using added CSV coordinates. It overlays a second variable—consumption-based emissions—to contrast regional footprints.
