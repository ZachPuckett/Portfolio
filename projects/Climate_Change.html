Mapping Projected Climate Change in the United States

This project uses raster datasets to model projected changes in climate variables across the US, emphasizing the spatial variability of climate impacts.

The final map visualizes absolute change in projected winter precipitation (2071–2090 vs. 1975–2005) and identifies geographic patterns related to climate vulnerability.

Methodology

  • Loaded a raster dataset showing projected absolute change in winter precipitation (1975–2005 vs. 2071–2090), sourced from the U.S. Forest Service's National Forest System Climate Change dataset.
  • Overlaid state boundaries with transparent fill and labeled each using the state name field.
  • Changed the projection to a suitable conic projection for the contiguous US (Albers Equal Area).
  • Added an inset map zoomed to the Pacific Northwest for localized analysis.
  • Included scale indicators and a legend to enhance interpretability and readability.

Geoprocessing in QGIS included coordinate system transformations, adjusting raster symbology with linear interpolation, and combining raster layers with vector overlays to communicate projected spatial change. All visualizations were finalized in QGIS Layout Manager using multi-frame map elements, labeling tools, and custom projection settings.

Data Source: U.S. Forest Service, Rocky Mountain Research Station — National Forest System Climate Change Maps

Map 1: Mixed Symbol Map – Cumulative vs 2019 CO₂ Emissions

This map combines a choropleth layer of cumulative CO₂ emissions with a dot density overlay of 2019 emissions. The map uses an equal-area projection to preserve spatial density and reveal long-term contributors versus current emitters.

Map 2: Orthographic Map – PM2.5 Exposure in Africa

This map shows average exposure to PM2.5 air pollution using an orthographic projection centered on New Delhi. The custom projection highlights regional health risks from air pollution.

Map 3: Regional CO₂ Emissions – Proportional Symbol Mapping

This proportional symbol map displays cumulative emissions by region using added CSV coordinates. It overlays a second variable—consumption-based emissions—to contrast regional footprints.
